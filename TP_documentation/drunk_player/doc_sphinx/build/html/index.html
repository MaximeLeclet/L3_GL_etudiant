Drunk_player_cli¶

Contents:

Description¶

Drunk_player_cli permet de lire les vidéos d’un dossier et de générer un fichier vidéo après avoir trop bu.

Utilisation¶

usage: ./drunk_player_cli.out <folder> <output file>

Exemple¶

$ ./drunk_player_cli.out ../data/ drunk_video.avi elapsed time: 5.444s
